# Monero 거래소 선택 가이드

> XMR을 거래하는 거래소를 고르고, 입금 주소를 받고, XMR 이름을
> 사칭하는 스캠 토큰을 피하는 방법.

---

## 왜 이렇게 어려운가

Monero (XMR)는 강력한 내장 프라이버시를 가진 몇 안 되는 주요
암호화폐 중 하나예요. 그래서 자주 표적이 돼요:

- **규제 압력.** 일부 국가와 거래소가 XMR을 상장폐지하거나,
  특정 사용자로 제한하거나, 입출금을 비활성화했어요.
- **스캠 토큰.** Monero의 브랜드가 인식 가능하기 때문에,
  Solana, Ethereum, 기타 체인에 가짜 "XMR" 또는 "MONERO" 토큰이
  수십 개 존재해요. Monero가 아니에요.

이 페이지는 진짜 거래소를 고르고, 진짜 XMR 주소를 받고, 함정을
피하도록 도와줘요.

## "Monero" 항목이 진짜인지 확인하는 법

가장 중요한 단계. 어떤 거래소에서든 입금 페이지에서 "XMR" 또는
"Monero"를 검색하면 **여러 항목**이 나타나요. 대부분은 스캠
토큰이에요. 진짜는 다음 속성으로 식별 가능:

### 진짜 Monero (XMR) 입금

| 속성         | 진짜 XMR                                              |
| ------------ | ----------------------------------------------------- |
| 이름         | `Monero` 또는 `Monero (XMR)`                          |
| 네트워크     | `Monero` 또는 `XMR` ("ERC20", "BEP20", "Solana" 아님) |
| 티커         | `XMR`                                                 |
| 가격         | CoinGecko와 일치: ~$300 (시장에 따라 변동)            |
| 주소 형식    | 95~106자, `4` 또는 `8`로 시작                          |
| 일 거래량    | $50M+ (진짜)                                          |

### 가짜 XMR 토큰 (스캠)

| 속성         | 가짜 XMR 토큰 (Solana / Ethereum 등)                 |
| ------------ | ----------------------------------------------------- |
| 이름         | `XMR`, `Wrapped Monero`, `MONERO` (단어만)           |
| 네트워크     | `Solana`, `Ethereum`, `BSC`, `Base`                  |
| 티커         | `XMR` 또는 `WXMR` 또는 `MONERO`                      |
| 가격         | $0.00001 ~ $0.10 (진짜 XMR과 크게 다름)              |
| 주소 형식    | 32~44자, `0x` (Ethereum) 또는 `6`/`7`/`8`/`9`/`A`/`B`/`C`/`D`/`E`/`F`/`G`/`H`/`J`/`K`/`L`/`M`/`N`/`P`/`Q`/`R`/`S`/`T`/`U`/`V`/`W`/`X`/`Y`/`Z` (Solana) |
| 일 거래량    | $0 ~ $50K (매우 작음)                                  |

> **주소 형식이 결정적인 테스트입니다.** 진짜 Monero 주소는
> **95자 이상**이고 `4` 또는 `8`로 시작해요. 복사한 주소가
> 60자 미만이거나 `0x`나 `4`/`8` 외의 숫자로 시작하면,
> **Monero가 아니에요**. 그 주소로 채굴하지 마세요. 수익은 잃어요.

## 단계별: XMR 입금 주소 받기

이 흐름은 모든 거래소에서 본질적으로 동일해요. 버튼 이름은 정확히
일치하지 않지만 순서는 같아요.

```
  +------------------------------------------------------------+
  |  1단계:  회원가입                                           |
  |          - 이메일 또는 휴대폰 번호                           |
  |          - 강한 비밀번호 (대문자 + 소문자 + 숫자 + 특수문자,   |
  |            8자 이상)                                         |
  +------------------------------------------------------------+
                              |
                              v
  +------------------------------------------------------------+
  |  2단계:  KYC (Know Your Customer) 완료                     |
  |          - 신분증 (여권, 운전면허증 등)                      |
  |          - 셀카 또는 짧은 영상                               |
  |          - 주소 (때로 자동)                                  |
  |          - 1~5분 소요, 때로 24시간까지                        |
  +------------------------------------------------------------+
                              |
                              v
  +------------------------------------------------------------+
  |  3단계:  2단계 인증 (2FA) 활성화                            |
  |          - 강력 추천: TOTP (Google                          |
  |            Authenticator, Authy)                            |
  |          - 최고: 하드웨어 보안 키 (YubiKey)                   |
  |          - 2FA 백업 코드를 안전한 곳에 저장                  |
  +------------------------------------------------------------+
                              |
                              v
  +------------------------------------------------------------+
  |  4단계:  입금 페이지로 이동                                  |
  |          - 보통:  지갑  ->  입금                             |
  |          - 또는:  자산  ->  받기                              |
  +------------------------------------------------------------+
                              |
                              v
  +------------------------------------------------------------+
  |  5단계:  Monero 찾기                                        |
  |          - 검색:  XMR  또는  Monero                         |
  |          - 위 비교표로 진짜를 식별                            |
  +------------------------------------------------------------+
                              |
                              v
  +------------------------------------------------------------+
  |  6단계:  네트워크 확인                                      |
  |          - 유효한 유일한 네트워크는 "Monero" 또는 "XMR"      |
  |          - 네트워크 선택기가 보이면 "XMR" 선택               |
  |          - "ERC20", "BEP20", "Solana" 절대 선택 금지        |
  +------------------------------------------------------------+
                              |
                              v
  +------------------------------------------------------------+
  |  7단계:  입금 주소 복사                                     |
  |          - 95~106자                                          |
  |          - 4 또는 8로 시작                                   |
  |          - 복사 버튼 사용 (수동 선택은 글자 빠뜨리기 쉬움)    |
  +------------------------------------------------------------+
                              |
                              v
  +------------------------------------------------------------+
  |  8단계:  xmrig-config.json에 붙여넣기                      |
  |          - 풀 설정의 "user" 필드                             |
  |          - 앞뒤 공백 OK; XMRig이 자름                        |
  +------------------------------------------------------------+
```

## 가입 후 거래소가 XMR을 비활성화하면?

이건 일어날 수 있어요. 일부 거래소가 가입 후, 입금 전에, 새로운
사용자 또는 특정 국가의 사용자에 대해 XMR을 비활성화해요.

**할 일:**

1. 이미 입금한 자금이 있으면 출금 (보낸 곳으로 다시)
2. 거래소에 법정화폐가 있으면 은행으로 출금
3. 위 표의 다른 거래소에 계정 개설
4. **모든** 거래소가 XMR을 비활성화했다면, [docs/getting-started.ko.md](getting-started.ko.md)의 본인 지갑 경로 사용

## 실수로 가짜 주소로 채굴한 경우

잘못된 주소를 XMRig 설정에 붙여넣고 잠시 돌렸다면, XMR은 사라졌어요.
블록 보상은 가짜 토큰의 스마트 컨트랙트에 갔고, 당신에게 가지 않았어요.
복구 방법 없어요.

**좋은 소식:** XMRig을 몇 분만 돌렸다면 거의 아무것도 벌지 못했을 거예요
(데스크톱 CPU 채굴은 시간당 1센트 미만).

**교훈:** XMRig 시작 전에 항상 주소 형식 확인 (95자 이상, `4`/`8` 접두사).

## 빠른 참조: 진짜 vs 가짜

```
  진짜 XMR                              가짜 XMR (사용 금지)
  --------------------                 -------------------------
  Monero (XMR)                          XMR
  네트워크: Monero                       네트워크: Solana
  시작: 4... 또는 8...                  시작: 6..., 7..., 8..., 9...
  길이: 95~106자                        길이: 32~44자
  가격: ~$300                            가격: $0.0001
  거래량: $50M+                          거래량: <$5K
```

의심되면 주소를 Monero 블록 익스플로러(`xmrchain.net` 같은)에
**채굴하기 전에** 붙여넣으세요. 진짜 주소는 잔액 페이지(또는 새
주소면 "거래 없음")를 보여줘요. 가짜 주소(사실 Solana 지갑)는 에러를
내거나 완전히 다른 체인의 데이터를 보여줘요.
