# XMR(Monero)란 무엇인가

> Monero의 역사, 기술, 비교, 실사용, FAQ를 다루는 완전 가이드.
> 채굴을 시작하기 전에 알아두면 좋은 XMR의 모든 것.

---

## 한 문단 요약

**Monero(XMR)는 2014년 출시된 프라이버시 중심 암호화폐**로,
비트코인의 투명한 블록체인과 달리 **모든 거래의 송신자·수신자·금액을
기본적으로 숨겨요.** Bitcoin Cash의 하드포크인 Bytecoin에서 분기했고,
**"발행 한도가 없다"**는 점이 비트코인(2,100만 개 한도)과 가장 큰
차이예요. 오늘날 다크웹 결제와 규제 회피의 상징이 됐지만, 정당한
프라이버시 권리(자선 기부, 노조, 정치 난민 등)도 많아요.

  │                                                        │

  │            └─> 깨끗한 체인에서 새로 시작                 │
  │            └─> 이름: Esperanto로 "coin"               │
  │                                                        │
  │  2017.01  RingCT 도입 (거래 금액 암호화)                │
  │                                                        │
  │  2018.10  Kovri 프로젝트 시작 (네트워크 익명성)         │
  │                                                        │
  │  2019.10  RandomX 알고리즘 (CPU 친화)                   │
  │            └─> ASIC 저항성 강화, GPU 마이닝 약화         │
  │                                                        │

**핵심 마일스톤**:
- **2017 RingCT** — 거래 금액을 암호화해 "누가 얼마 보냈는지"도 숨김
- **2019 RandomX** — ASIC 채굴기를 못 쓰게 만들어 일반인이 CPU로도 참여 가능 (이 가이드의 XMRig)

## 기술 — 어떻게 프라이버시를 구현하나

Monero의 마법은 3가지 기술의 결합:

### Ring Signatures (링 서명) — "누가 보냈나" 숨기기

```
  실제 발신자: Alice

  Ring Signature로 생성된 서명 그룹:
  ┌─────────────────────────────────────────────┐
  │  Alice   ◀─── 실제 발신자 (검증자도 모름)    │
  │  Bob                                         │
  │  Carol                                       │
  │  David                                       │
  │  Eve                                         │
  └─────────────────────────────────────────────┘

  외부 관찰자: "누군가 5명 중 한 명이 보냈음" (확률 20%)
  수신자만 알 수 있음: "Alice가 나한테 보냄"
```

### Stealth Addresses (은닉 주소) — "누가 받았나" 숨기기

```
  발신자 Bob이 Alice에게 1 XMR 송금:

  Bob이 아는 것: Alice의 공개 주소 (view key로 파생 가능)
  블록체인에 기록: 일회용 1회용 주소 (예: 5AbCd...9xYz)

  Alice의 지갑만: 자기 view key + spend key로 실제 입금 자동 인식
  외부 관찰자: "누군가 5AbCd...9xYz가 받았음" (Alice인지 모름)
```

### RingCT (Ring Confidential Transactions) — "얼마나" 숨기기

```
  공개 블록체인: amount = ??? (Pedersen commitment로 암호화)
  네트워크 합의: amount = 0이 아니고, 음수 아니고, double-spend 아님 검증

  외부 관찰자: "누군가 1 XMR 비슷한 양이 오감" (정확한 금액 모름)
```

### RandomX (작업 증명) — CPU 친화 알고리즘

```
  Bitcoin (SHA-256):    ASIC이 100배 빠름, GPU가 10배 빠름
  Ethereum (Ethash):    GPU 우세, ASIC 등장
  Monero (RandomX):     CPU와 RAM을 무작위로 실행
                        → ASIC/GPU 우위 2배 이하
                        → 일반 데스크톱 CPU로 채굴 가능
                        → 노트북/스마트폰도 채굴 가능
```

> **왜 중요한가?** Bitcoin은 채굴이 산업화돼서 "51% 공격"이 ASIC
> 보유자 손에 있음. Monero는 일반 사용자도 채굴에 참여 가능 →
> 탈중앙화 유지.

## 다른 코인과의 비교

### Bitcoin vs Monero

| 항목 | Bitcoin (BTC) | Monero (XMR) |
|------|---------------|---------------|
| **목적** | 디지털 금 (가치 저장) | 프라이버시 화폐 (익명 결제) |
| **거래 투명성** | 완전 공개 (누가, 누구에게, 얼마) | 모두 숨김 |
| **발행 한도** | 2,100만 개 (2140년 채굴 종료) | **무제한** (약 0.6 XMR/분 영원히) |
| **블록 시간** | 10분 | 2분 |
| **채굴 알고리즘** | SHA-256 (ASIC 전용) | RandomX (CPU 친화) |
| **에너지 효율** | 낮음 (채굴 전쟁) | 높음 (일반 PC로 가능) |
| **합법성** | 대부분의 국가 합법 | 🇺🇸🇰🇷 일부 거래소 상장폐지 (규제 압박) |
| **거래 추적** | 가능 (체인분석) | 이론적으로 불가 |

### Monero vs 다른 프라이버시 코인

| 코인 | 출시 | 프라이버시 기법 | 거래 속도 | 채굴 가능 | 활성도 |
|------|------|----------------|-----------|-----------|--------|
| **Monero (XMR)** | 2014 | Ring sig + Stealth + RingCT | 2분 | ✅ (RandomX) | ⭐⭐⭐⭐⭐ |
| Zcash (ZEC) | 2016 | zk-SNARKs (선택적) | 75초 | ✅ (Equihash) | ⭐⭐⭐ |
| Dash | 2014 | PrivateSend (믹싱, 선택) | 2.5분 | ✅ | ⭐⭐ |
| Pirate Chain (ARRR) | 2018 | zk-SNARKs (강제) | 1분 | ✅ | ⭐⭐ |
| Verge (XVG) | 2014 | Tor + I2P | 30초 | ✅ | ⭐ |

> **Monero의 차별점**: 다른 프라이버시 코인들은 **"선택적"**
> (기본은 투명)인 반면, Monero는 **"강제"** (모든 거래가 자동
> 프라이버시). 사용자가 "잘못"해서 프라이버시 끄는 게 불가능.

### Monero vs Tornado Cash (이더리움 믹서)

| 항목 | Monero | Tornado Cash |
|------|--------|--------------|
| **플랫폼** | 자체 체인 | 이더리움 위 앱 |
| **신뢰 가정** | 수학적 (암호학) | 스마트 컨트랙트 (2022 미국 제재) |
| **사용자** | 누구나 | 메타마스크 사용자만 |
| **제재 가능성** | 체인 자체 (어려움) | 스마트 컨트랙트 주소 (제재 가능) |

> **Monero의 우위**: Tornado Cash는 2022년 미국 재무부가 제재해
> 사용자가 받는 입력이 막혔어요. Monero는 분산된 풀 노드가 전
> 세계에 있어 단일 제재가 어려움.

## 자주 묻는 질문 (FAQ)

**Q: XMR이 비트코인을 대체하나요?**
A: 아니요. 다른 목적이에요. BTC는 "디지털 금" (가치 저장), XMR는
"디지털 현금" (프라이버시 결제). 둘 다 공존.

**Q: 왜 거래소들이 XMR을 상장폐지하나요?**
A: EU의 TFR(자금세탁방지규정)과 미국 FinCEN 규제가 강화되면서,
거래소가 자체적으로 "고위험 자산"을 제거하는 추세. 2024년
Binance가 EU에서 XMR 신규 상장 중단한 것이 대표적.

**Q: XMR은 추적이 정말 불가능한가요?**
A: **이론적으로는 매우 어렵지만, 불가능은 아니에요.** 2020년
Chainalysis가 Monero 추적 도구를 발표했지만 실효성은 논쟁 중.
"대부분의 사용자 추적은 불가능"이 정확한 표현.

**Q: XMR을 채굴하면 법적으로 문제가 되나요?**
A: 한국에서 채굴 자체는 합법 (전력 사용은 신고 필요할 수 있음).
채굴한 XMR을 매각/사용하는 건 일반 자산 거래로 과세 대상.

**Q: 발행량에 상한이 없으면 인플레이션이 심한 거 아닌가요?**
A: **완화 인플레이션** (disinflation) 구조 — 2024년 기준 약
0.87%/년. 시간이 갈수록 발행량이 줄어들어 비트코인처럼 0에
수렴. 그래서 "암호화폐의 인플레이션 헷지"로 부르기도.

**Q: 21세기 금본위제와 비슷한가요?**
A: 의견이 갈려요. 호평론자는 "BTC는 디지털 금, XMR는 디지털 현금,
둘 다 필요", 비평론자는 "정부가 결국 규제할 것".
